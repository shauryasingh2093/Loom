Design additional product screens for the Loom app, a speculative wellness tool that visualizes a user's mind, body, and environment as threads that weave together.

The visual system should use threads that tangle and untangle instead of graphs or dashboards.

Maintain the same soft calming aesthetic, warm neutral background, flowing thread visuals, and accessible typography.

The design should follow WCAG accessibility guidelines with clear contrast and readable text.

The UI should feel like a real product similar to Duolingo or Headspace — calm, friendly, and easy to use.

Do NOT create dashboard-style charts.

Everything should be expressed through thread movement and weaving metaphors.

Add These Screens
1. Bottom Navigation System

Add a persistent bottom navigation bar with four sections.

Navigation tabs:

Loom (home)
Threads (signals and insights)
Guide (AI companion)
Weave (progress and history)

Use simple icons and labels.

The active tab should have a subtle glow or thread highlight.

2. Daily Weave Progress Screen

Create a screen that shows how the user's threads changed throughout the day.

Title:
Your Weave Today

Visualization:
Thread pattern timeline from morning → evening.

Examples:
Morning – tangled threads
Afternoon – stabilizing threads
Evening – smoother weave

Add text insight like:

"Your weave began stabilizing after your breathing reset."

Include a small progress indicator like:

Today's balance: 42%

3. Pattern Insights Screen

Create a screen where Loom explains patterns it has noticed.

Title:
Your Patterns

Examples of insights:

• Noise tends to tangle your weave during study sessions
• Your body threads tighten after long meetings
• Your weave stabilizes after stepping outside

Visualize this using subtle thread diagrams, not graphs.

Each insight appears as a card with animated threads.

4. Calm State Screen

Create a state where the user's weave is fully balanced.

Title:
Your Weave is Calm

Threads appear smooth and evenly spaced.

Add a message like:

"Your threads are flowing smoothly. Your system is balanced."

Include a button:

Return to your weave

5. Privacy & Safety Screen (Safeguards Requirement)

Create a Privacy & Safety page to address ethical safeguards.

Title:
Your Data & Safety

This screen should contain four sections:

Data Privacy
Explain that Loom processes signals locally and does not share emotional data.

User Control
Allow options to pause sensing, delete data, or reset personal insights.

Shared Space Awareness
Explain that Loom never analyzes or identifies other people.

Emotional Safety
Explain that Loom provides guidance, not diagnosis.

Use calm icons and simple toggle controls.

Add a message:

"Your weave belongs only to you."

6. AI Companion Identity Screen

Create a screen introducing the AI Companion.

Title:
Meet Loom

Show a small animated thread knot icon that untangles as the user follows suggestions.

Example message:

"I help you notice when your threads begin to tighten and guide you back to balance."

Visual Design Rules

Use flowing thread animations as the primary visual language.

Examples:

Calm → smooth flowing threads
Stress → tangled threads
Recovery → threads slowly untangle

Use soft colors:

sage green
muted coral
lavender
warm beige

Maintain generous spacing and readable typography.

Avoid dense UI or dashboard layouts.
